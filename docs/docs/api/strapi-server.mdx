# Strapi Server

:::info

Current state: **Stable**

:::

The Strapi server module permits to generate a Strapi http server.

## Module: Strapi server

### `createServer(strapi)`

- `strapi`: [<Type>Strapi</Type>](Strapi)
- Returns: <Type>StrapiServer</Type>

```javascript
const server = createServer(strapi);

server.listRoutes();
```
